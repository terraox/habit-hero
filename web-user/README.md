# Habit Hero - User Portal

A mobile-first React web client for Habit Hero users.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS (Dark mode first)
- **State Management**: React Query (@tanstack/react-query)
- **Routing**: React Router DOM
- **Charts**: Recharts
- **Icons**: Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5174](http://localhost:5174) in your browser.

## Features
- **Dark/Light Mode**: Toggle via the top bar.
- **Dashboard**: View daily progress and today's habits.
- **Habit Tracking**: Mark habits as complete.
- **Calendar**: View habit history.
- **Responsive**: Optimized for mobile viewports.
